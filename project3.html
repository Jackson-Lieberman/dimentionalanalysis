Dimensional Analysis Calculator

A tool for converting between metric unit prefixes, built by Jackson Lieberman.

Project Overview

The Dimensional Analysis Calculator is a tool I built for my chemistry class to easily convert between different metric prefixes.
Users can input a value and select starting and target prefixes to quickly get the correct converted value.

Features

  • Convert between any metric prefixes (kilo, centi, milli, micro, etc.).
  • User-friendly interface for inputting values and selecting prefixes.
  • Instant calculation and result display.

What I learned

  • How to build an interactive tool with Python completely unaided.
  • Improved my understanding of dimensional analysis and metric prefixes.
  • Learned how to validate user input and handle conversions.
  • Practiced creating a responsive and clean design.
